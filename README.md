# Status Manager Project

## Course

COMP 484

## Description

This project demonstrates the use of the Document Object Model (DOM) and Browser Object Model (BOM) to create an interactive web page.

## Features

* Toggle status visibility
* Dynamic timestamp creation
* Styled list items using JavaScript
* Background color changes based on state
* Flashing control panel using `setInterval` and `clearInterval`

## Technologies Used

* HTML
* CSS
* JavaScript
